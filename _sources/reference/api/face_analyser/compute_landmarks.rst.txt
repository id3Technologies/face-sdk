===============================================================================
FaceAnalyser.computeLandmarks Method
===============================================================================

Module: :ref:`FaceAnalyser<face_analyser_class>`

Definition
----------

Computes 68 landmarks of a detected face.

.. important:: Loading the FaceLandmarksEstimator2A model is required to use this function.
