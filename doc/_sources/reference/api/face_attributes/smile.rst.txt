===============================================================================
FaceAttributes.smile Property
===============================================================================

Structure: :ref:`FaceAttributes<face_attributes_struct>`

Definition
----------

Value indicating whether the person is smiling or not.

100 means that the person is definitely smiling.

.. hint:: The recommended value is less than 75.
