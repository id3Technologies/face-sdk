===============================================================================
FaceAnalyser.segmentFace Method
===============================================================================

Module: :ref:`FaceAnalyser<face_analyser_class>`

Definition
----------

Computes face segmentation mask for background removal.

.. important:: This methods requires the FaceSegmenter1A model to be loaded.
