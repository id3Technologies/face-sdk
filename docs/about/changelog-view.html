Version 9.5.0 (2023-03-21)#

  • Added Face Detector 4A

  • Added new Portrait Processor module used to facilitate the creation of portraits in various scenarios such as enrolment of face verification. The module performs the following tasks:

    • Face detection and tracking with prediction models

    • Landmark detection

    • Head pose estimation

    • Subjects position verification

    • Template encoding and updating

    • Additional optional computations (age estimation, background uniformity computation, emotion detection, eye gaze detection, eye opening detection, face mask detection, gender estimation, glasses detection, hat detection, make-up detection, mouth opening estimation, occlusion detection, photographic attributes computation, presentation attack detection, red-eye detection, smile detection)

    • Verification of compliance on many quality attributes

    • Unified quality score computation

  • Fixed left/right eye in occlusions

  • Added method to update a FaceTemplate

  • Added FaceAnalyzer.computePoseWithLandmarks() method for more accurate pose estimation.

  • Added DetectedFace.getExpandedBounds() method.
