===============================================================================
PortraitProcessor.sharpnessThreshold Property
===============================================================================

Module: :ref:`PortraitProcessor<portrait_processor_class>`

Type: int

Definition
----------

Sharpness threshold. 

If the sharpness score is above this value, the image sharpness is sufficient.

.. hint:: Default value is 50.
