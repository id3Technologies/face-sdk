===============================================================================
Image.fromYuvPlanes Method
===============================================================================

Module: :ref:`Image<image_class>`

Definition
----------

Imports the image object from YUV planes.

            unsigned char * yPlane,

            unsigned char * uPlane, 
            int uPlaneSize, 
            unsigned char * vPlane, 
            int vPlaneSize, 
            int yWidth, 
            int yHeight, 
            int uvPixelStride, 
            int uvRowStride, 
            id3FacePixelFormat eDstPixelFormat)

      **Parameters**

        .. line-block::

          ``hImage`` :ref:`ID3_FACE_IMAGE<image_class>`
          :guilabel:`in` Handle to the Image object.

          ``yPlane`` unsigned char *
          :guilabel:`in` Buffer containing the Y plane.

          ``yPlaneSize`` int
          :guilabel:`in` Size of the 'yPlane' buffer.

          ``uPlane`` unsigned char *
          :guilabel:`in` Buffer containing the U plane.

          ``uPlaneSize`` int
          :guilabel:`in` Size of the 'uPlane' buffer.

          ``vPlane`` unsigned char *
          :guilabel:`in` Buffer containing the V plane.

          ``vPlaneSize`` int
          :guilabel:`in` Size of the 'vPlane' buffer.

          ``yWidth`` int
          :guilabel:`in` Width in pixels of the Y plane.

          ``yHeight`` int
          :guilabel:`in` Height in pixels of the Y plane.

          ``uvPixelStride`` int
          :guilabel:`in` Pixel-level stride in pixels of the U and V planes.

          ``uvRowStride`` int
          :guilabel:`in` Row-level stride in pixels of the U and V planes.

          ``eDstPixelFormat`` :ref:`id3FacePixelFormat<pixel_format_enum>`
          :guilabel:`in` The destination pixel format to convert the input image to.


      **Returns**

        .. line-block::

          ``int``
          An error code.
          See :ref:`id3FaceError<face_error_enum>` file for the list of possible error codes.
