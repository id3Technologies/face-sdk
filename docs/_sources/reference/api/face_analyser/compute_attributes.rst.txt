===============================================================================
FaceAnalyser.computeAttributes Method
===============================================================================

Module: :ref:`FaceAnalyser<face_analyser_class>`

Definition
----------

Computes the attributes of a detected face.

.. important:: Loading the model FaceAttributesClassifier2A is required to use this function.
