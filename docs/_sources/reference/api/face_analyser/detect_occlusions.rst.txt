===============================================================================
FaceAnalyser.detectOcclusions Method
===============================================================================

Module: :ref:`FaceAnalyser<face_analyser_class>`

Definition
----------

Detects the presence of occlusions on top of a detected face.

.. important:: Loading the FaceOcclusionDetector1A model is required to use this function.
