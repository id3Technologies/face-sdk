===============================================================================
FaceAttributes.mouthOpen Property
===============================================================================

Structure: :ref:`FaceAttributes<face_attributes_struct>`

Definition
----------

Value, from 0 to 100, indicating whether the subject's mouth is open or not.

.. hint:: The recommended value is less than 35.
