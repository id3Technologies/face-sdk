Image.set Method#

Module: Image

Definition#

Sets the image object fields. This function does not allocate any memory and is unsafe! The user must handle the pixels pointer carefully.

int id3FaceImage_Set (
    ID3_FACE_IMAGE hImage,
    int width,
    int height,
    id3FacePixelFormat ePixelFormat,
    void * pixels)

Parameters

hImage ID3_FACE_IMAGE
in Handle to the Image object.

width int
in Width in pixels of the image.

height int
in Height in pixels of the image.

ePixelFormat id3FacePixelFormat
in The pixel format of the image.

pixels void *
in Pointer to the pixels of the image. Unsafe!

Returns

int
An error code.
See id3FaceError file for the list of possible error codes.
public static Image Set(int width, int height, PixelFormat pixelFormat, IntPtr pixels)

Parameters

width int
in Width in pixels of the image.

height int
in Height in pixels of the image.

pixelFormat PixelFormat
in The pixel format of the image.

pixels IntPtr
in Pointer to the pixels of the image. Unsafe!

Returns

Image
The newly created image.

Exceptions

FaceException
An error has occurred during Face Library execution.
