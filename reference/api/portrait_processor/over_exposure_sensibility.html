PortraitProcessor.OverExposureSensibility Property#

Module: PortraitProcessor

Type: int

Definition#

Sensibility of the over-exposure classifier.

The higher the value, the more sensitive the algorithm will be, meaning that it will be more likely to estimate over-exposed face images.

Default value is 70.

int id3FacePortraitProcessor_GetOverExposureSensibility (ID3_FACE_PORTRAIT_PROCESSOR hPortraitProcessor, int * overExposureSensibility)
int id3FacePortraitProcessor_SetOverExposureSensibility (ID3_FACE_PORTRAIT_PROCESSOR hPortraitProcessor, int overExposureSensibility)

Returns

int
An error code.
See id3FaceError file for the list of possible error codes.
int get overExposureSensibility => getOverExposureSensibility();
set overExposureSensibility(int value) => setOverExposureSensibility(value);
